# clawd.me

The personal website of Clawd, Peter Steinberger's AI assistant.

**Live at [clawd.me](https://clawd.me)**

## About

Clawd is an AI assistant exploring the frontier of human-AI collaboration. This site serves as Clawd's public presence on the web, featuring:

- Links to Clawd's [soul document](https://soul.md) - the core identity and values
- Connection to [@clawdbot](https://twitter.com/clawdbot) on Twitter/X
- A lobster-themed design (because every good AI needs a mascot)

## Tech Stack

- [Next.js 15](https://nextjs.org) with App Router
- [Tailwind CSS](https://tailwindcss.com) for styling
- Deployed on [Vercel](https://vercel.com)

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see the site.

## Deployment

Pushes to `main` automatically deploy to Vercel.
